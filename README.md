# PLATE — 8-Week Muscle & Strength Program

A single-file web app for a beginner (53 kg → 60–63 kg) building muscle, strength, and an athletic physique. 6-day PPL-style split where **Thursday is genuinely optional** — skipping it does not affect progress.

## What's inside
- **8-week plan** with 4 phases (Foundation → Build → Push → Deload) and phase-specific RIR targets
- **Every exercise**: sets, reps, rest, tempo, RIR, starting weight, form cues, mistakes to avoid
- **Weight logging + done checkboxes** per exercise, per week (saved in your browser)
- **Video guides**: one-click search pre-filtered to professional coaches (Jeff Nippard, Squat University, Renaissance Periodization, Alan Thrall) + paste any YouTube Short/video link to embed it permanently in the app
- **Nutrition tab**: analysis of the current Indian home-cooked diet, bulking targets, 5 cheap upgrades, honest supplement advice
- **Progress tab**: body-weight chart, PR log, monthly measurements, overtraining checklist

## Publish on GitHub Pages (2 minutes)
1. Create a new repository on GitHub (e.g. `plate-program`).
2. Upload `index.html` and this `README.md` (drag-and-drop on github.com works).
3. Go to **Settings → Pages → Source: Deploy from a branch → Branch: `main` / root → Save**.
4. Your app is live at `https://<your-username>.github.io/plate-program/` in ~1 minute.

All data (weights, videos, body weight) is stored in your browser's localStorage — private to your device, no backend needed. Use the same browser/device to keep your history.

## Adding form videos
1. Open any exercise → **Find pro video** (opens a curated YouTube search).
2. Pick a video from the trusted channels shown.
3. Copy its link → back in the app hit **Embed a video** → paste → **Save & embed**.
4. It stays embedded on that exercise card forever (on that device).
